# Rock Paper Scissors

A small, single-page Rock Paper Scissors web game built with HTML, CSS and JavaScript.

## Features
- Clickable buttons for Rock, Paper, and Scissors
- Keyboard shortcuts: `R` (rock), `P` (paper), `S` (scissors)
- Live score tracking for Player and Computer
- Reset button to clear scores
- Small, responsive UI with subtle animations

## How to run

1. Open the project folder in File Explorer.
2. Double-click `index.html` or from PowerShell run:

```powershell
start index.html
```

The game runs entirely in the browser—no server or build steps required.

## Controls
- Click one of the choice buttons.
- Or press `R`, `P`, or `S` on your keyboard to play quickly.
- Click the **Reset** button to set both scores back to zero.

## Files
- [index.html](index.html) — main page and UI
- [style.css](style.css) — styles and responsive layout
- [script.js](script.js) — game logic and interactivity

## Customize
- Change the UI in `style.css` to match your colors.
- Modify `script.js` to add features like best-of, score limits, or persistent scores.
